# 🧮 Java Console Calculator

## 🎯 Objective

This project is built as part of a **Java Developer Internship** to understand the basics of Java programming such as syntax, methods, loops, conditionals, and input handling.

The goal was to create a **simple calculator** that performs basic arithmetic operations via the console using Java.

---

## 🧠 Key Concepts Used

- Java **methods** for each arithmetic operation (add, subtract, multiply, divide)
- **Scanner** for input handling
- **Loops** to allow repeated calculations
- **Conditionals** for menu control
- Basic **error handling** for divide-by-zero

---

## 📋 Features

- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division (with divide-by-zero check)  
- 🔁 Loop to allow multiple calculations until user exits

---

## 🛠 Tools & Technologies

- Java (JDK 23 or higher)
- VS Code / IntelliJ IDEA CE
- Terminal / Command Prompt
